Represents a joystick on the controller. 

This struct is used to represent a joystick on the controller. It contains two values, x and y, to represent the position of the joystick. Since we want to output joystick positions as ASCII, we store the x and y values as perentages of maximum values. For example, a stick all the way left would be -\/100\% along the x-\/axis, and a stick all the way up would be 100\% along the y-\/axis. Each stick also has a unique string identifier of length 2 to represent the stick in ASCII. \Hypertarget{con__state_8h_ab61ffbd6da6f72e48c1701d3d35b2063}\index{con\_state.h@{con\_state.h}!Trigger\_t@{Trigger\_t}}

Represents a trigger on the controller. 

This struct is used to represent a trigger on the controller. It contains a value to represent the position of the trigger. Since we want to output trigger positions as ASCII, we store the trigger value as a percentage of the maximum value. For example, a trigger pressed all the way down would be 100\%. Each trigger also has a unique string identifier of length 2 to represent the trigger in ASCII.
