\doxysubsection{Detailed Description}
Represents the state of the Google Stadia controller. 

This struct is used to represent the state of the Google Stadia controller. It contains all the buttons, joysticks, triggers, and D-\/pad of the controller, with all of the unique identifiers for each control on the controller. The identifiers go as follows\+:
\begin{DoxyItemize}
\item D-\/pad\+: DP
\item Joysticks\+:
\begin{DoxyItemize}
\item LJS
\item RJS
\end{DoxyItemize}
\item Triggers\+:
\begin{DoxyItemize}
\item LTR
\item RTR
\end{DoxyItemize}
\item Letter buttons\+:
\begin{DoxyItemize}
\item LAB
\item LBB
\item LXB
\item LYB
\end{DoxyItemize}
\item Other buttons\+:
\begin{DoxyItemize}
\item LTB (Left Digital \doxylink{struct_trigger}{Trigger})
\item RTB (Right Digital \doxylink{struct_trigger}{Trigger})
\item RSB (Right stick button)
\item LSB (Left stick button)
\item STB (Stadia button/ on/off button)
\item MEN (Menu button)
\item CPT (Capture button)
\item GAS (Google Assistant button)
\item OPT (Option button)
\item RBP (Right bumper)
\item LBP (Left bumper) 
\end{DoxyItemize}
\end{DoxyItemize}
