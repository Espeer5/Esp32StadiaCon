\doxysubsection{Detailed Description}
Represents a D-\/pad on the controller. 

This struct is used to represent a D-\/pad on the controller. It contains a compass direction to represent the position of the D-\/pad. The D-\/pad also has a unique string identifier of length 2 to represent the D-\/pad in ASCII.
